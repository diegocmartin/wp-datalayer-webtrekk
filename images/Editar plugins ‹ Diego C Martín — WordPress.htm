Documentación de la edición de plugins

Documentación sobre cómo escribir plugins

Soporte

Puedes usar el editor para realizar cambios a cualquier archivo php de tus plugins. Cuidado si realizas cambios, la actualización de los plugins producirá que tus modificaciones sean sobrescritas y se pierdan.

Elige un plugin a editar en el menú superior derecha y haz clic en el botón Seleccionar. Haz clic una vez sobre cualquier nombre de archivo para cargarlo en el editor. No olvides guardar tus cambios (Actualizar archivo) cuando acabes.

El menú de documentación bajo el editor contiene un listado de funciones PHP detectadas en el archivo del plugin. Al hacer clic en Buscar te lleva a una página web con información sobre esa función concreta.

Cuando se usa un teclado para navegar:

  • En el área de edición la tecla Tab introduce un carácter de tabulación.
  • Para salir de este área pulsa la tecla Esc seguida de la tecla Tab.
  • Usuarios con lector de pantalla: al estar en el modo de formulario puede que necesites pulsar la tecla Esc dos veces.

Si quieres hacer cambios pero no quieres que se sobreescriban cuando se actualice el plugin, deberías pensar en escribir tu propio plugin. Para obtener información acerca de cómo editar plugins, escribir desde cero el tuyo, o simplemente entender mejor su anatomía, echa un vistazo a los enlaces de aquí abajo.
